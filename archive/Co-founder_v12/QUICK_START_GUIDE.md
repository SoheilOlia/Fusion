### **Method 1: Quick Install**
```bash
# Copy to global Cursor directory
cp -r Cofounder_v11 ~/.cursor/cofounder-gpt-v11/

# Create global command
echo '#!/bin/bash\ncat ~/.cursor/cofounder-gpt-v11/ChatGPT_10_Files/COFOUNDER_CHATGPT_MASTER_PROMPT_V11_COMPLETE.md' > ~/.cursor/bin/cofounder-gpt
```
